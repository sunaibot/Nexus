## 启动所有服务进行网页交互测试

### 计划概述
启动后端服务、前台应用和后台管理系统，让您可以进行完整的网页交互测试。

### 执行步骤

#### 1. 清理现有进程
- 停止所有现有的 node 进程，避免端口冲突

#### 2. 启动后端服务
- 目录：`e:\NOWEN-main\apps\server`
- 命令：`npm run dev`
- 端口：8787
- 功能：提供 API 服务、数据库管理、用户认证等

#### 3. 启动前台应用
- 目录：`e:\NOWEN-main\apps\frontend`
- 命令：`npm run dev`
- 端口：5173
- 功能：展示书签、分类、搜索等前台功能

#### 4. 启动后台管理
- 目录：`e:\NOWEN-main\apps\manager`
- 命令：`npm run dev`
- 端口：5174
- 功能：管理书签、分类、设置等后台功能

#### 5. 验证服务状态
- 检查所有服务是否正常启动
- 确认没有错误日志

#### 6. 提供访问信息
- 前台访问：http://localhost:5173
- 后台访问：http://localhost:5174
- 后台登录账号：
  - 用户名：admin
  - 密码：admin123

### 预期结果
- 三个服务全部正常运行
- 可以正常访问前台和后台页面
- 可以正常登录后台管理系统
- 可以进行书签的增删改查操作
